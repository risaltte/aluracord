# aluracord
Projeto criado na Imersão React 01/2022 com @alura @omariosouto @peas

Foram adicionadas as funcionalidades:
- Botão de enviar mensagens.

- Deletar mensagens: O usuário pode apagar somente as suas próprias mensagens, mensagens vazias não são enviadas.
  

- Além do realtime ao adicionar uma novame mensagem, também foi adicionado realtime ao deletar a mensagem.

- Logout
